My Megi Guide

功能輔助說明

依照主要工作流程整理：登入、上傳、辨識審核、聯絡人、API Token、用戶管理與 API 文件。

登入與版本資訊

進入系統前先登入。登入卡片下方會顯示目前服務版本，正式環境請使用管理員建立的帳號。

名片上傳

正面至少一張，背面選填。支援拖拉或點選上傳，送出後會自動執行 OCR 與 LLM 結構化辨識。

最近匯入與審核

最近匯入分成未審核與已審核。辨識完成後可進入審核表單，低信心度資料由人工確認。

聯絡人查詢

可用姓名、公司、Email、電話與分類搜尋。列表每頁 20 筆，右側提供檢視、編輯與刪除操作。

API Access Token

用戶與內容管理員可建立第三方整合 Token。每個帳號最多一組 Active token，重新產生會讓舊 token 過期。

用戶管理

系統管理員可搜尋、新增、編輯、啟用與停用用戶。用戶列表每頁 20 筆，權限由角色控制。

我的資訊

登入後在 Top Banner 下方可看到目前用戶。右側齒輪可修改顯示名稱與密碼，登入帳號不可修改。

Swagger API 文件

開發者可從 Swagger 查詢 API schema、request/response 格式與測試端點，CLI 也使用同一組 HTTP API。
